# GitHub user worker - небольшой проект по поиску профилей на GitHub

##  Установка проекта и запуск
```
$ git clone https://github.com/IvanXab/github-users-worker.git
```
```
$ npm ci
```
```
$ npm run dev
```

### Использованые технологии:
- Vue 3
- Pinia
- Element-plus
- SCSS
- Axios
